# What Day of week is this day web app

Web application using nuxt 3 to solve the question “What weekday is today?”

## Setup

Make sure to install the dependencies:

```bash
# yarn
yarn install

# npm
npm install

# pnpm
pnpm install
```

## Development Server

Start the development server on http://localhost:3000

```bash
npm run dev
```
